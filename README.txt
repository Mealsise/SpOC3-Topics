Setup Instructions
Install Conda: Make sure you have Anaconda or Miniconda installed.

Create the Environment: Run this command to set up the environment:
    conda env create -f environment.yml

Activate the Environment:
    conda activate satellite_optimisation

Start Jupyter Notebook (Optional):
    jupyter-notebook



experiments_large_dro.ipynb: for running experiments for the large dro

experiments_large_halo.ipynb: for running experiments for the large halo

results_generation.ipynb: for generating results for the large dro and large halo
for use in the presentation and paper
